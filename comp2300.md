### **Applied Cryptography**

Cryptology encompasses two fields:
- Cryptography - The development of cryptographic techniques
- Cryptanalysis - The breakup of cryptgraphic techniques

Types of ciphers:
- Symmetric ciphers
- Asymmetric ciphers
- Hash functions

Plaintext = Original format of message
Ciphertext = Encrypted form of message
Encryption algorithm = Algorithm transforms plaintext to ciphertext
Decryption algorithm = Algorithm transforms ciphertext to plaintext


## Security properties
- Confidentiality = protection of information from unauthorized access
- Integrity = protection from intentional or accidental unauthorized changes
- Authentication of origin
- Entity authentication = provides timless guranantees, involves real-time verification of claimed identity
- Non repudiation = prevents claims of incorrect or non-identity
- Anonyimity = action cannot be linked to your real identity
- Avaliability = provision of service in predictable manner
- Fairness = same outcome for all parties

Kerckhoff's Principle = A cryptosystem should be secure, even if everything about the system, excluding the key, are public knowledge

Black box security
- Keep algorithms secret, usually have a weakness and will be discovered and exploited

Crystal/Glass box security
- Publish your algorithms to community review, weak algorithms will not survive

Adversary Capabilities:
- Passive attacker = can read packets but not modify them
- Active attacker = can remove, reorder, reinsert and modify packets

Quantifying security:
- The strength of a cryptosystem is expressed by the amount of work required to crack it
- The *work* factor is generally a function of the size of the cryptographic key
- Ideally, cryptosystem is secure if any attack requires an unfeasible amount of time if the key is larger than a specific size
- Reality, cryptosystem is secure if all known attacks requires an unfeasible amount of time if the key is larger than a specific size

# Greatest Common Divisor
An integer d is called a common divisor of a and b, if d | a and d | b.\n
If all other common divisors of a and b divide d, and d is non-negative, then d is called the greatest common divisor (gcd) of a and b.
